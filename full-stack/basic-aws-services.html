Basic AWS Services

AWS has several services but you don’t need to know all of them. Here are the most common ones that you can get started with:

  • EC2 - Virtual machines in the cloud
  • VPC - Virtual private cloud
  • S3 - Object storage
  • Route 53 - DNS
  • SES - Email sending
